Monika-Bot

Monika-bot is a discord bot that I created in order for give my friends and I more interactivity with each other within our discord server. The bot is coded in Python and includes some fun features such as:

  • The ability to call on 'Steam Db' API's to gather real-time data on PC games hosted on the Steam servers.
  • Using public MyAnimeList API's has the ability to recommend currently airing anime based on their popularity and user ratings.
  • interactivity with Mudae, another popular Discord bot, to allow users to more easily track each members participation to the 'Collection' game that Mudae offers.

Tech Blog

This tech blog was part of an assignment from my Georgia Tech Bootcamp which utilized JavaScript, HTML/CSS, and Handlebars to help create templates based on our JS functions. The blog has an established connection to a MySQL database utilizing bcrypt to ensure sensitive user data is encrypted and stored properly. This tech blog was used among our classmates and allowed them to do the following:

  • Create a unique username using a given username and password which kept track of your posts and comments.
  • Create posts on a public board which anyone else on the website could view and comment on.
  • Post comments under fellow classmates blog posts to let them know what your take on their opinion might be.

MEMEnto

MEMEnto was a collaborative project with fellow classmates to create a fun "meme" template for users to create and post their meme creations with one another. This project utilizes JavaScript for all of its logic and express js specifically to handle the API and MySQL database connections, which were all hosted on a public Heroku server. This web application allowed users to:

  • Create their own unique meme's using the applications template tools
  • Download their creations to share across their personal devices and take on the go.
  • Share their creations on the main page of the website for other users to view and even download for themselves.
